# Kubernetes Production Observability Stack

Bu dizin, Kubernetes production ortamları için:

- Prometheus (Metrics)
- Grafana (Visualization)
- Loki (Centralized Logging)
- Alertmanager (Alerting)

bileşenlerinden oluşan **tam kapsamlı gözlemlenebilirlik mimarisini** içerir.

## Amaç

- Servis sağlığını gerçek zamanlı izlemek
- Performans darboğazlarını erken tespit etmek
- Olası kesintileri önceden öngörmek
- Log ve metrikleri merkezi yönetmek
